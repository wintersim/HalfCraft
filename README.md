# HalfCraft
Simple LibGdx Minecraft clone I made for school. (Comments are in german)

## Features

* Infinite random terrain
* Different Biomes
* Blocks can be broken
* New Blocks can be placed
* Simple Hotbar
* 'Creative Mode' Only

## Keybinds
| Move              | W A S D     |
|-------------------|-------------|
| Look              | Mouse       |
| Break Block       | LMB         |
| Place Block       | RMB         |
| Fly               | F           |
| Switch Item       | Scrollwheel |
| Debug render-mode | B           |
| Debug reset world | R           |
